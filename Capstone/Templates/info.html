About this web application.

This web application was built primarily for data exploration of the powerlifting dataset available on Kaggle. Users can navigate to 'Visualize' to explore data in the interactive dashboard. Users can also go to predict and get feedback based on users queries.

This site was is build using Python Flask and Dash. The predictive features are implemented in Tensorflow.
